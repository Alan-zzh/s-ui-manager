# S-UI v1.4.1 全自动配置项目

## 项目说明
本项目用于全自动配置S-UI面板v1.4.1，支持以下协议：
- VLESS Reality（端口443）
- VLESS WS（端口8443，支持CDN）
- Trojan WS（端口8444，支持CDN）
- Hysteria2（端口4433，支持端口跳跃）

## 文件列表
- `SUI全自动配置_正确版.py` - 完整数据库配置脚本
- `S-UI_Bug文档.txt` - 详细Bug记录与修复过程

## 使用方法
1. 修改脚本中的敏感信息（IP、密码、密钥等）
2. 运行脚本自动配置S-UI面板
3. 查看Bug文档了解常见问题和解决方案

## 注意事项
- 所有敏感信息已用占位符替换（YOUR_XXX）
- 使用前请替换为实际值
- 详细Bug记录请参考 `S-UI_Bug文档.txt`

## 技术栈
- S-UI v1.4.1
- Sing-Box v1.13.4
- SQLite数据库
- Python 3
